| | |
|---|---|
| Quem avalia | Sede Nestlé |
| Como | Dash de Ruptura |
| Período | Mensal |

| Pontos | Faixa |
|---|---|
| **10,0** | Ruptura **igual ou abaixo de 5%** para os SKUs Hero |

**Dupla validação — as duas precisam ser satisfeitas:**
1. Ruptura de no máximo **5%**; e
2. Das lojas indicadas para roteirização, pesquisas realizadas em **pelo menos
   60% dos PDVs**.

- Origem: pesquisas dos promotores do Broker via **Connect Merchan**; apuração no
  **Dash de Ruptura Zero 3.0**. Acompanhado na Sede, acessível a qualquer momento
  pelo **CND**.
- *"A cada trimestre, será enviada uma nova pesquisa de ruptura, ajustada com os
  SKUs Heros atuais."*
